# A simple registration and login form with Laravel, MySQL and Vue
## Backend

### Migrate migrations

```
php artisan migrate
```

### Run the server

```
php artisan serve
```

### Running the MySQL server

```
docker-compose up -d
``` 
to start
```
docker-compose down
``` 
to stop
